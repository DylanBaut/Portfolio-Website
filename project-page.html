Radar Respiratory Monitoring,
CS 437: Topics in Internet of Things

Created wireless breathing rate monitoring system using raw ADC data from Radar. Used Doppler-range profile to extract phase information from single phase bin and graphically display torso micromovements related to breathing. Displayed inhale/exhales on RaspberryPi LED, and send UDP message if halt in breathing occurs.

For our Midterm, we found hidden spy camera's by packet sniffing over wifi to capture the RSSI values of an emitting motion-detecting camera. Using these signal strength values, we mapped out competition area in a 2D space using joystick input, IMU, and step detection. After graphical analyis, estimations are made to where the spy camera most likely is.

Tetris FPGA Implementation
ECE 385: Digital Systems Laboratory

Replicated the classic NES game of Tetris through our SOC design and software code. Used a USB keyboard as input and a VGA display as output, we implemented all the basic functions of stacking blocks, clearing rows, score counting, and game ending. A NIOS II CPU is used alongside a RAM component for on-chip storage, with most of the screen logic being written in C software with VGA controls based in the AVL interface module.

FLying Object Tracking Program
CS 445: Computational Photography

Given a tracking shot of flying object in motion, return a trajectory of object overlayed over stabilized background. Used MATLAB to train darknet Object detection network on flying objects such as helicopters, airplanes, etc. Converted video to homographic overlay of key frames to map out panoramic background, and use detector to trace trajectory over image.
